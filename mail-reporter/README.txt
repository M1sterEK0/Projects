Оповещение по Email. Может прикреплять файлы к письму. Может работать по расписанию.


Настраивается через config.py

Ничего не меняйте, просто измените адрес в переменной To_Email на тот, на который хотите получить сообщение.

Само сообщение, можно редактировать в переменной message в файле app.py(пока что для теста именно переменная служит для отображения простого сообщения)

Так же к письму привязывается(вложением) файл ./res/file.txt, содержимое данного(file.txt) файла можете менять как угодно.
